## Prerequisites

- cmake >= 3.0.0
- g++ 
- gfortran
- openscenegraph 3.6.5, openscenegraph-qt 3.5.7
- GSL (libgsl-dev on Ubuntu 22.04)
- GLM (libglm-dev on Ubuntu 22.04)
- Qt5

## Building

To compile vecds run `./build.sh`

To run vecds with a GUI, go to the bin directory and run:

`$ cd bin`
`$ ./vecds`
